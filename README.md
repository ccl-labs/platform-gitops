# platform-gitops
Platform Engineering portfolio — GitOps management repository.

## 概要
ArgoCDのソースとなるリポジトリ。プラットフォーム全体の「あるべき姿」を管理する。

## ディレクトリ構成
| ディレクトリ | 内容 |
|---|---|
| `bootstrap/` | ArgoCDのRoot App（App of Apps） |
| `platform/` | 基盤コンポーネント（Ingress / Monitoring / Logging / Secrets / Policy） |
| `apps/` | サービス一覧 |
